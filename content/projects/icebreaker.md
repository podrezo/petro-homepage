# Icebreaker

Icebreaker is a lightweight Chrome extension that delivers over 400 fun and engaging questions right to your browser's sidebar. Perfect for spicing up team meetings, virtual standups, and corporate events with questions that get everyone laughing, sharing, and bonding as a team.

The extension transforms those awkward "how's everyone doing?" moments into genuine team bonding experiences with instant access to entertaining questions. Whether you're navigating through the curated collection or hitting "Random" for spontaneous team moments, it's designed to energize meetings with minimal setup required.

You can find the source code at [podrezo/icebreaker](https://github.com/podrezo/icebreaker) on GitHub or in the chrome web store [here](https://chromewebstore.google.com/detail/icebreaker/impckacciegdbcnfpjihpdkclolbjida?fbclid=IwY2xjawM9hHhleHRuA2FlbQIxMABicmlkETFqWWFONHZXZTdnalY1WkJxAR6-tKgACGWRdPgLkGabCzM09R_uGbwqoTli6n-CIc8DC0We2BXOxFX6EvRuCA_aem_N-Ut5PhWYH3zQ0Iz8kAihw).
